The Itemization Overhaul Codex

Every system in the suite, explained in depth - with the underlying numbers and worked examples.

Itemization Overhaul replaces vanilla quality with a Diablo-style loot engine, and eight optional addons build on top of it. This wiki documents each mechanic on its own page. Start with the Core Overview, then work down the sidebar, or jump to whatever you are tuning.
