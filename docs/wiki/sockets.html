Sockets & Runes

How many sockets an item rolls, the rune ladder, and the slot-dependent effects.

Socket roll

Non-set gear has a 40% chance (tunable) to roll any sockets at all; the count is then weighted by gear slot. The roll is final - nothing changes it afterward, not re-rolls, not crafting, not Isekai's forge.

Gear slotSocketsCount weights
Weapon1 - 440 / 30 / 20 / 10

Helmet1 - 350 / 35 / 15
Other gear1 - 270 / 30

One rune, four effects

Each rune grants exactly one static effect per gear slot. Stat effects ride the same equipped-gear pipeline as Core affixes, so they stack with everything. On-hit procs (ignite / stun / lifesteal) are melee-only and can be toggled in settings.

Socketing

Select an item with a free socket and press Socket rune…. The float menu lists every rune on the map, each already showing the exact effect it would grant this item. The nearest capable colonist fetches the rune and carves it in; the rune is consumed. Socketing is permanent - no removal, no swaps.

Worked example. You drop Brim into a 3-socket greatsword (10% ignite on hit), then a Hark (+10% melee damage), then an Ix (5% stun). The same three runes in a chestplate would instead give heat armor, damage reduction, and blunt armor - so plan your runes around the slot.
