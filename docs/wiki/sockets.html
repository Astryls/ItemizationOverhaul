Sockets & Runes

How many sockets an item rolls, the rune ladder, and the slot-dependent effects.

Socket roll

Non-set gear has a 40% chance (tunable) to roll any sockets at all; the count is then weighted by gear slot. The roll is final - nothing changes it afterward, not re-rolls, not crafting, not Isekai's forge.

Gear slotSocketsCount weights
Weapon1 - 440 / 30 / 20 / 10
Body armor1 - 445 / 30 / 17 / 8
Helmet1 - 350 / 35 / 15
Other gear1 - 270 / 30

One rune, four effects

Each rune grants exactly one static effect per gear slot. Stat effects ride the same equipped-gear pipeline as Core affixes, so they stack with everything. On-hit procs (ignite / stun / lifesteal) are melee-only and can be toggled in settings.

Socketing

Select an item with a free socket and press Socket rune…. The float menu lists every rune (and gem) on the map, each already showing the exact effect it would grant this item. The nearest capable colonist fetches the socketable and carves it in; it is consumed. Socketing is permanent - no removal, no swaps.

Drag-and-drop socketing True RPG Inventory

With True RPG Inventory installed, socketing becomes tactile. Open a colonist's RPG inventory doll and simply drag a rune or gem onto a socketed item — every valid target lights up with a socket affordance as you drag, and dropping carves the socketable in on the spot.

  • Works everywhere in the panel. Drop onto an item sitting in the inventory grid, or straight onto an equipped piece — body armor, helmet, the main weapon, the off-hand, or a sidearm slot.
  • Live target highlighting. Only items with a free, compatible socket highlight, so you can't waste a rune on the wrong piece.
  • Same rules underneath. Drag-drop calls the exact same socketing logic as the float menu — permanent, one socketable per socket, effect determined by the item's slot.
No mod juggling. The bridge is one-directional and reflection-based: it drives Runes & Socketables' socket API from the inventory UI, and quietly does nothing if either mod is absent.
Worked example. You drop Brim into a 3-socket greatsword (10% ignite on hit), then a Hark (+10% melee damage), then an Ix (5% stun). The same three runes in a chestplate would instead give heat armor, damage reduction, and blunt armor - so plan your runes around the slot.
