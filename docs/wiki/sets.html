How sets drop

Any item that rolls affixes has a chance (default 12%) to belong to a named set - but only on Legendary or Relic rarity, so set pieces are always top-tier. A set item drops branded with its set name in green. Because sets need Legendary+, plain and White items are never set pieces, honouring the Core drop rate.

Wearing a set

Equip matching pieces and bonuses stack as you go - 2, 3, and 4-piece thresholds, then a capstone aura when the full set is worn. The aura pulses every few seconds within its radius: a nova deals damage to enemies, a ward heals allies. The health tab tracks your progress as <Set Name> (X/4). See Bonuses & Auras for the full breakdown.

Suite integration

  • Runes - set items get a standardized socket count per set and slot.
  • Stats & Levels - Legendary/Relic set pieces naturally demand high levels.
  • Relic sets - a set made of relic-rarity pieces gains an extra +25% to all bonuses.
Worked example. You assemble all four Cataphract's Vigil pieces. The 2- and 3-piece armor bonuses are already active; the fourth completes Iron Sanctuary, a healing ward that mends nearby colonists each pulse. If those four pieces were relic-rarity, every bonus is 25% stronger and the health tab reads 4/4 ✦.
